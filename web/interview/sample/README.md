[写一个三栏布局](3-cloumn-layout-MLR.html)
=========================================
这个题目经常见到。这里的这个题目给出了html结构

```html
<body>
  <div>
    <div id="M"></div>
    <div id="L"></div>
    <div id="R"></div>
  </div>
</body>
```
其中M是中间栏，要自适应宽度。L是左侧栏，R是右侧栏，宽度都是固定像素，比如200px。
写出相应的css
